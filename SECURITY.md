# Security Policy

`formsmith` is pre-beta and repo-first.

Please do not include private source code, credentials, proprietary examples, or
secret values in public issues. Reduce reports to the smallest public Clojure or
ClojureScript form that reproduces the behavior.

For now, report suspected unsafe rewrites, crashes, or sensitive-output behavior
through GitHub Issues. Maintainers may ask for a smaller public reproduction
before investigating.
