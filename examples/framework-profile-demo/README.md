# Framework Profile Demo

Tiny source tree used to exercise framework profile detection and framework
contracts.

It is not a runnable app. It exists so `formsmith profiles` can demonstrate
framework facts without depending on external repos.

- `ui.cljs` covers Reagent and re-frame.
- `modern.cljs` covers HSX and RFX contract-style guidance.
- `mobile.cljd` covers ClojureDart and the visible Flutter `f/nest` rewrite.
