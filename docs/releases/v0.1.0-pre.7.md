# v0.1.0-pre.7

`v0.1.0-pre.7` expands the framework-aware surface and adds the first
ClojureDart-specific UI rewrite.

## Install Coordinate

```clojure
{:deps {io.github.M-simplifier/formsmith
        {:git/url "https://github.com/M-simplifier/formsmith.git"
         :git/tag "v0.1.0-pre.7"
         :git/sha "decbd33310ada782d9097c195284a1ad9dcd4e39"}}}
```

Clojure CLI requires the `:git/sha`; tag-only coordinates are rejected.

## Changes Since v0.1.0-pre.6

- Add `.cljd` discovery to the source file set.
- Add source-scan namespace dependency extraction so framework profiles can see
  ClojureDart string requires such as `"package:flutter/material.dart"`.
- Add framework profiles for HSX, RFX, and ClojureDart.
- Add HSX/RFX `llm-refactor` contracts for Reagent local ratom state and
  implicit one-argument RFX dispatch calls.
- Add `clojuredart/widget-child-chain`, a default-safe rewrite that flattens
  long Flutter `.child` chains into `cljd.flutter/nest` when the current
  `cljd.flutter` alias is explicit.
- Add report-only guidance for deprecated `cljd.flutter.alpha` and
  `cljd.flutter.alpha2` requires.
- Tolerate ClojureDart type metadata such as `^#/(m/Animation double)` during
  traversal instead of crashing sexpr-dependent Clojure rules.
- Add public framework-profile docs and a ClojureDart site demo.
