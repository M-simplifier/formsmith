# v0.1.0-pre.6

`v0.1.0-pre.6` is a safety release for reader-sensitive macro-template source
forms found while preparing an external re-frame adoption candidate.

## Install Coordinate

```clojure
{:deps {io.github.M-simplifier/formsmith
        {:git/url "https://github.com/M-simplifier/formsmith.git"
         :git/tag "v0.1.0-pre.6"
         :git/sha "89a06b941e32bb9fe78e5fab22d005a5147234b8"}}}
```

Clojure CLI requires the `:git/sha`; tag-only coordinates are rejected.

## Changes Since v0.1.0-pre.5

- Treat syntax-quote, unquote, and unquote-splicing as reader-sensitive
  contexts.
- Prevent default rewrites from rewriting inside macro templates where a
  grouping `do` can be semantically required for unquote-splicing.
- Make the pinned open-source field-trial runner ignore Formsmith's own repo
  config so cloned `target/field-trials` workspaces cannot be hidden by
  `ignore-paths`.
- Keep field-trial reports public-safe by recording repository-relative finding
  paths.
