# v0.1.0-pre.3

`v0.1.0-pre.3` is a versioned GitHub source release of `formsmith`.

## Install Coordinate

```clojure
{:deps {io.github.M-simplifier/formsmith
        {:git/url "https://github.com/M-simplifier/formsmith.git"
         :git/tag "v0.1.0-pre.3"
         :git/sha "9649e96a0f60c585d9f68608648f63839f5395af"}}}
```

Clojure CLI requires the `:git/sha`; tag-only coordinates are rejected.

## Changes Since v0.1.0-pre.2

- Protected autofix against source forms that carry reader metadata, comments,
  or reader-discard structure.
- Updated metadata-sensitive regression coverage so aggressive seq rewrites do
  not drop Hiccup-style key metadata.
- Kept the public rule surface unchanged while tightening when rewrites are
  allowed to apply.

## Adoption Note

The `clj-ops-ledger` adoption trial exposed a metadata-sensitive aggressive
rewrite risk before the release coordinate was merged into that app. This
release is the safety tightening that made that adoption clean:

```text
clojure -M:formsmith check src test
files=8 changed=0 findings=0

clojure -M:formsmith fix --check --aggressive src test
files=8 changed=0 findings=0
```

See [docs/adoptions/clj-ops-ledger.md](../adoptions/clj-ops-ledger.md).
