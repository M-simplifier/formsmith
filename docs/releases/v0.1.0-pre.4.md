# v0.1.0-pre.4

`v0.1.0-pre.4` is a robustness release for reader-sensitive source forms found
during public open-source field trials.

## Install Coordinate

```clojure
{:deps {io.github.M-simplifier/formsmith
        {:git/url "https://github.com/M-simplifier/formsmith.git"
         :git/tag "v0.1.0-pre.4"
         :git/sha "a2449935781fc6f3b2b4451aa3e8e30e0bbc9abf"}}}
```

Clojure CLI requires the `:git/sha`; tag-only coordinates are rejected.

## Changes Since v0.1.0-pre.3

- Skip rewrites inside reader-discard and reader-conditional contexts.
- Treat rewrite-clj unsupported `sexpr` conversion failures as rule no-matches
  instead of process crashes.
- Add protected corpus and tests for reader-discard source.
