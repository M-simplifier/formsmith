# v0.1.0-pre.2

`v0.1.0-pre.2` is a versioned GitHub source release of `formsmith`.

## Install Coordinate

```clojure
{:deps {io.github.M-simplifier/formsmith
        {:git/url "https://github.com/M-simplifier/formsmith.git"
         :git/tag "v0.1.0-pre.2"
         :git/sha "9b1b4fdca1a045ef79495ce7e94106b570f00368"}}}
```

Clojure CLI requires the `:git/sha`; tag-only coordinates are rejected.

## Changes Since v0.1.0-pre.1

- Added syntax-safe `get/keyword-lookup`: `(get m :k)` -> `(:k m)`.
- Added basic and protected corpus coverage for keyword lookup rewrites.
- Updated aggressive `get` + `seq` corpus expectations so keyword lookup cleanup
  composes with existing seq rewrites.
