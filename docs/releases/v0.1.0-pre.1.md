# v0.1.0-pre.1

`v0.1.0-pre.1` is the first versioned GitHub source release of `formsmith`.

## Supported Use

- consume `formsmith` from Clojure CLI as a pinned git dependency
- run `check` as a no-write CI gate
- run conservative `fix` locally for default autofixes
- use `baseline` and `.formsmith.edn` when introducing the tool to existing repos
- use `contracts` output as structured work for humans or AI agents

## Not Yet Supported

- no Clojars package
- no editor extension
- no standalone binary
- no claim of broad architecture-level refactoring
- no claim that aggressive rewrites should be required for every project

## Evidence Included In This Release

- local test and corpus verification through `bb ci`
- repo-only first-run validation through `bb validate-cold-start`
- public GitHub Actions CI and GitHub Pages workflows
- a separate public full-stack Clojure pressure app consuming `formsmith` as a git dependency:
  <https://github.com/M-simplifier/formsmith-pressure-app>

## Upgrade Notes

This is a pre-beta source release. Pin the release tag, and pin the release
commit SHA as well when your dependency tooling or production policy requires
immutable git coordinates.
