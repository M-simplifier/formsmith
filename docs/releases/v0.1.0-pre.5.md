# v0.1.0-pre.5

`v0.1.0-pre.5` is a patch-quality release for `cond` catch-all rewrites found
while preparing the first external adoption PR packet.

## Install Coordinate

```clojure
{:deps {io.github.M-simplifier/formsmith
        {:git/url "https://github.com/M-simplifier/formsmith.git"
         :git/tag "v0.1.0-pre.5"
         :git/sha "4bd1d7228aebf24a0cc7b80c83c84396ea7d1fbc"}}}
```

Clojure CLI requires the `:git/sha`; tag-only coordinates are rejected.

## Changes Since v0.1.0-pre.4

- Preserve the line break before catch-all branches when rewriting two-clause
  `cond` forms to `if`.
- Trim trailing spaces after replacing a multiline `cond` head.
- Add regression coverage from the Kaocha candidate adoption shape.
